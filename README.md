# Task_3
Temperature Converter – Oasis Infobyte Internship (Task 3)

This project is a simple and interactive Temperature Converter built using HTML, CSS, and JavaScript as part of my virtual internship at Oasis Infobyte. The converter allows users to input a temperature, choose the unit, and instantly view the converted result.

Features

1.Input field with number validation

2.Unit selection using dropdown or radio buttons (Celsius ↔ Fahrenheit)

3.“Convert” button to trigger real-time conversion

4.Display area for the final converted temperature

5.Optional added support for Kelvin conversion

Tech Stack

1.HTML5

2.CSS3

3.JavaScript

4.VS Code
